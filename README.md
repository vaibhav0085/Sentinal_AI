# Sentinel AI
Futuristic Cybersecurity Platform Landing Page & Dashboard.

## 🚀 Overview
Sentinel AI is a cutting-edge web interface designed for a next-generation cybersecurity platform. The project features a stunning "Dark Mode" aesthetic, glassmorphism elements, and interactive 3D-style visualizations to convey security, precision, and innovation.

## ✨ Features
- **Immersive UI/UX**: Deep midnight blue background with neon accents (Cyan, Green, Purple).
- **Glassmorphism Design**: Modern, translucent card effects with background blurring.
- **Interactive Visuals**:
  - Animated "Cyber Shield" with rotating rings and pulsing core.
  - Interactive particle background on the landing page.
- **Dashboard Interface**:
  - File Drop Zone for malware scanning simulation.
  - Real-time progress visualization.
  - Threat Score Speedometer.
  - Responsive sidebar and layout.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile screens.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid, Keyframe animations.
- **JavaScript**: Canvas manipulation for particles, DOM manipulation for dashboard interactivity.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: Inter & Roboto for clean, modern typography.

## 📂 Project Structure
```

```

## 🚀 How to Run
1.  Simply open `index.html` in any modern web browser to view the Landing Page.
2.  Click "Launch Detector" or navigate to `dashboard.html` to view the Dashboard.

## 🎨 Design Philosophy
The design embodies "Military-grade security meets Silicon Valley innovation", utilizing a specific color palette:
- **Background**: `#0a0f1b` (Deep Dark Blue)
- **Accents**: `#00f2ea` (Cyan), `#00ff9d` (Green), `#9d00ff` (Purple)
